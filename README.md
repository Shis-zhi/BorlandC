## 路径说明

`HZK`：汉字库相关文件

`include`：头文件

`info`：存放数据文件

`source`：工程源码

**注意：源码中所有路径均采用绝对路径，如需自定义，请自行修改，否则，请在 `BC31/DISK_C`文件夹内，也就是 `BORLANDC`的同级目录新建一个名为 `mine` 的文件夹，并将repo内的所有文件移动至此文件夹内。**

*附：建议将源码内所有的的绝对路径改为相对路径*
